# FoundrOS

The Autonomous Startup Operating System.

## Vision
FoundrOS is an AI-native operating system that creates a virtual startup organization. Instead of one AI assistant, FoundrOS creates an executive team composed of specialized autonomous agents.

## Goals
To simulate how a real startup operates by allowing agents to collaborate, disagree, negotiate, critique, assign work, plan, execute, and continuously improve startup ideas.

## Architecture
- **Communication:** Event-driven
- **Memory:** Hybrid (Private Notebooks + Shared Wiki)
- **Planning:** Hierarchical with negotiation
- **Prompts:** File-based, separate from code

## Tech Stack
- **Language:** Python 3.12
- **Package Manager:** uv
- **Formatter/Linter:** Ruff
- **Testing:** pytest
- **Docs:** MkDocs
