# Future Improvements

This document tracks features intentionally deferred from Version 1 of the Prototype to maintain MVP focus.

## Deferred to v2+
- Event-driven publish/subscribe agent communication (EventBus).
- Long-term memory storage (e.g., Vector DBs for organizational wiki).
- Knowledge Graph implementation.
- LangGraph / MCP integration.
- Advanced Agent Scheduling and Background Workers.
- Database Optimization.
- Multiple Organization support.
- Multi-user / Auth support.
